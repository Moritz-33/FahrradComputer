# Fahrradcomputer

A bicycle computer project built with PlatformIO.

## Features

- Real-time speed measurement
- Distance tracking
- Trip statistics
- Ride duration tracking
- Odometer functionality

## Hardware Requirements

- Microcontroller (e.g., Arduino, ESP32)
- Speed sensor
- Display module
- Power supply

## Installation

1. Clone or download this project
2. Open in PlatformIO
3. Configure the board and port settings
4. Build and upload to your microcontroller

## Usage

Mount the sensor on your bicycle wheel and display on the handlebars. The computer will automatically track your ride data.
